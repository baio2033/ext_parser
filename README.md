# Usage

	python parser.py <image.dd>

# Description
	
	ext4 파일시스템을 해석해주는 파이썬 툴입니다.
	root directory 부터 순회가 가능하며 파일 추출 기능이 있습니다.(단, depth 가 0 인 파일에 한하여 추출이 가능함)
	Usage 에 나온대로 파서를 실행하고 원하는 디렉토리 또는 파일의 번호를 입력합니다.
